Poker Game Post

May 31, 2019

https://github.com/renazhong/poker_game.git

The task was to write a program that would allow a number of people (passed as an argument) to play a game of poker. We also had to ask what each player’s name was, and this was done through an input message.

The next step was to random draw a hand of 5 cards to each player of the game. It is important to note that since poker is drawing without replacement, we had to make sure that none of the cards were drawn twice. This was done through keeping a list of drawn cards (in tuples of rank and suit) and testing if the card was already in the list. If it was already drawn, then we would have to redraw. Using a for loop would not have worked because if the card was drawn the loop would still count it, so in this scenario, we used a while loop, and for the condition where the card drawn was not part of the drawn pile, we would increase the count, and when it reached 5, the loop would terminate.

For every player of the game, an empty list was created to represent their hand, and any card not part of the drawn list would be added the hand of 5 until it was full. This list was then added as the value for the dictionary that kept track of all the players and their hand of cards.

Then for every player, we went through their hand of cards (the values of the dictionary) and sorted each from highest to lowest value. Since there were strings as well as integers, we created another dictionary in order to keep the values uniform, changing the strings into integer equivalents.

This concluded the first part of the program, which was setting up the game. The next part is to determine who wins the game. The rules of this game was that whoever had the highest card in their hand would win, and if there was a tie, the second highest card between those with the tie would determine the winner.

Since all the cards were already listed from highest to lowest value, all that we needed to do was to find the largest value and get the player who held that card. This is done through a for loop that loops through the highest card of all the players and then returns the player with the largest card.
